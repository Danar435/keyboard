# Keyboard
A colemak xkb layout with modifier keys changed. Meant to be modified and used across devices. 

![Keyboard Layout](https://raw.githubusercontent.com/Danar435/keyboard/refs/heads/main/layout.png)

The layout specifically changes the positions of the modified keys when using 'shift', 'altgr' and 'shift + altgr'. The basic idea is to place the more common keys closer together, while retaining a somewhat familiar layout. For example, the question mark and exclamation mark were moved on top of the 'comma' and 'period' keys, rather than the 'slash' and '1' keys. AltGr is used as an additional layer, rather than being largely ignored. It gives access to some special symbols, and also gives me access to the extra danish characters "æ ø å". Combining it with shifts gives access to the most common greek characters, which might come in handy for mathematicians. 

A via layout is included for the OLKB Planck keyboard. It goes hand-in-hand with the layout, while also working okay with other layouts (though I wouldn't recommend it).
